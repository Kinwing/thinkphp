Think@PHP {$Think.version} 安装指南 [ 返回首页 ]

ThinkPHP本身不需要安装，只有示例需要安装，如果从来没有安装过ThinkPHP，请按照下面的步骤来安装才能演示示例。

1、上传解压后的目录和文件到服务器，如果你的服务器环境是Linux环境，请设置Home目录和Admin目录下面的Cache、Conf、Temp、Logs子目录的属性为可写（777）

2、创建数据库

可以参考下面的SQL语句来创建数据库：
CREATE DATABASE `thinkphp` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci; 使用uft8编码导入admin.sql 示例sql文件

注意：如果mysql版本在4.1以下，需要修改sql文件，以保证导入成功

3、配置数据库信息

打开框架根目录下的config.php文件，按照定义制定你所使用数据库的链接参数。


define('DB_TYPE','mysql'); // 数据库类型
define('DB_HOST','localhost'); // 数据库服务器主机
define('DB_NAME','thinkphp'); // 框架数据库名
define('DB_USER','root'); // 数据库服务器链接用户名
define('DB_PWD',''); // 数据库服务器链接密码
define('DB_PORT','3306'); // 数据库服务器链接端口号
define('DB_PREFIX','think'); // 数据库表前缀
到此，你已基本完成安装。打开浏览器输入地址，你就可以看到欢迎页面了，然后通过欢迎页面的链接就可以登录后台管理页面。
