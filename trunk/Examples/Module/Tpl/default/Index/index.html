ThinkPHP示例之：URL伪装

可以通过配置来实现URL的模块和操作的伪装
当前URL：
{$Think.server.PHP_SELF}
下面这些URL都指向 Index模块 的read 操作
正常URL：/Index/read/id/5
模块伪装：/Hello/read/id/5
操作伪装：/Index/a/id/5
模块和操作同时伪装：/Haha/a/id/5
示例源码
配置文件config.php
highlight_file(CONFIG_PATH.'config.php');
控制器IndexAction类
highlight_file(LIB_PATH.'Action/IndexAction.class.php');
