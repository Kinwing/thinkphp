ThinkPHP示例之：字段映射

要使用模型的自动创建create方法的话，ThinkPHP的表单名称就是数据表的字段名称，如果担心这样不够安全，可以定义字段映射，来隐藏实际的数据表字段名称。
下面的例子就运用了字段映射定义。请注意看表单名称和字段的对应关系。
