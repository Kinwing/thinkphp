日志处理类

支持下面的日志类型 WEB_LOG_DEBUG 调试信息 WEB_LOG_ERROR 错误信息 SQL_LOG_DEBUG SQL调试 分别对象的默认日志文件为 调试日志文件 systemOut.log 错误日志文件 systemErr.log SQL日志文件 systemSql.log
