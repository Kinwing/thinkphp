ThinkPHP示例之：视图模型

ThinkPHP的视图模型模拟了数据库的视图实现，可以方便地完成简单的数据关联操作。
注意看下面的数据列表中的类型名称就是通过视图模型来获取的。
{$vo.title} [ category:{$vo.category} time:{$vo.cTime|date='Y-m-d H:i:s',###} read:{$vo.readCount} | comment:{$vo.commentCount}]
