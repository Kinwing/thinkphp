ThinkPHP示例之：URL伪静态

可以轻而易举地通过配置来实现URL伪静态
当前URL：
{$Think.server.PHP_SELF}
实际操作 {$Think.const.MODULE_NAME}模块 的{$Think.const.ACTION_NAME} 操作
正常URL：/Index/read/id/5
伪静态URL：/Index/read/id/5.shtml
示例源码
配置文件config.php
highlight_file(CONFIG_PATH.'config.php');
控制器IndexAction类
highlight_file(LIB_PATH.'Action/IndexAction.class.php');
