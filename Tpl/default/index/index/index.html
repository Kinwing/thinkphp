Feature

FCS是一个快速、兼容而且简单的面向对象的超轻量级PHP开发框架，您可以容易地构建中小规模的应用。具有： 
快速：快速的应用开发和优化的执行速度

兼容：兼容PHP4和PHP5版本

简单：简洁的框架让你的应用变得更加简单

FCS框架用简洁的方式构建了复杂的应用框架，还会致力于发布丰富的类库组件和应用组件，不断地吸收和融入新的技术是FCS框架发展的宗旨，使你的应用易如反掌！

具体特性

  
1、基于MVC模式实现的面向对象的应用级开发框架

2、兼容PHP4.3以上（包括PHP5）
   
3、自定义错误和异常处理

4、简单的日志记录功能

5、模板动态缓存和页面缓存输出

6、模板文件和程序代码100％分离～

7、自定义的模板文件、缓存文件的后缀

8、静态文件生成及过期处理

9、支持自动模板编码转换和输出编码设置

10、方便灵活的应用组件定制和组装

11、对搜索引擎友好的支持
