ThinkPHP v1.5.0 API

ThinkPHP是一个快速、简单的面向对象的轻量级PHP开发框架，遵循Apache2开源协议发布，并且永久免费使用，是为了简化企业级应用开发和敏捷WEB应用开发而诞生的。借鉴了国外很多优秀的框架和模式，使用面向对象的开发结构和MVC模式，融合了Struts的Action思想和JSP的TagLib（标签库）、RoR的ORM映射和ActiveRecord模式，封装了CURD和一些常用操作，单一入口模式等，在模版引擎、缓存机制、认证机制和扩展性方面均有独特的表现。 使用ThinkPHP，你可以更方便和快捷的开发和部署WEB应用，任何PHP应用开发都可以从ThinkPHP的特性中受益。简洁、快速和实用是ThinkPHP发展秉承的宗旨，为此ThinkPHP会不断吸收和融入更好的技术以保证其新鲜和活力，提供WEB应用开发的最佳实践！
